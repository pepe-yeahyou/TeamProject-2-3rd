&nbsp;<< 메인페이지 기능 >>

1\. 로그인 시, 최초 보여지는 요약화면이다.

2\. 대시보드에서 해당 사용자의 프로젝트는 프로젝트 담당자이거나, co-worker일 경우에 대해서만 표시된다.

3\. 로그인한 팀원의 프로젝트 별 진척도를 계산하여 시각화하여 보여준다

4\. 진척도는 진행도에 따라 %로 표시된다

5\. 해당 프로젝트 클릭시, 상세 페이지로 넘어가게되며, 상세페이지에는 해당 페이지에 대한 상세 내용을 확인할 수 있다.



&nbsp;<< 입력페이지 >> 

1\. 프로젝트는 제목/내용/해야할 것을 작성한다.

2\. 한 프로젝트에 해야할 것은 여러개가 발생할 수 있다.

3\. 프로젝트 작성자는 co-worker를 배정할 수 있다.

4\. 해당 프로젝트의 데이터 첨부를 위해 파일 업로드 기능을 추가한다.



&nbsp;<< 상세페이지 >>

1\. 프로젝트의 제목/내용/진척도를 확인한다.

2\. 프로젝트를 수정/삭제할 수 있다.

2-1. 프로젝트 작성자만 진척도 관리를 진행한다(완료/진행중)

2-2. 프로젝트 co-worker는 진행중인 작업에 대해 완료/진행중 처리를 할수있다.

3\. 해당 프로젝트가 진행중일경우 채팅기능을 활성화 하며, 프로젝트가 종료될 경우 채팅기능은 비활성화된다.

4\. 채팅기능은 해당 프로젝트의 담당자와 co-worker만 진행할 수 있다.
